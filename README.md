# V - WEB GUI Testing
Tein yksinkertaisen web-sovelluksen, joka muuntaa väriarvoja kahteen suuntaan:
HEX → RGB
RGB → HEX
Frontend on staattinen HTML (public-kansio).
Backend on Node.js + Express.
Kun käyttäjä syöttää arvot ja painaa nappia, selain tekee GET-pyynnön backendille, joka palauttaa HTML-sivun tuloksella.
